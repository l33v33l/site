Notes à propos de HTML

<a>

L'Élément a est un ancre (anchor) vers un lien. href est un attribut. href = nom de l'attribut. Ce qui est entre les guillements = la valeur de l'attribut Un attribut modifie comment un élément fonctionne.
On peut avoir plusieurs attributs.

** Écrit dans un élément distinct ** L'attribut supplémentaire "target" avec "_blank" permet d'ouvrir le lien dans un autre onglet.
